# movie-ios-app
This is a prototype iOS mobile application to save and manage list of your favorite movies. 

## How to run simulator?
1. Let's run a server
https://github.com/sarahselee223/independent-study-backend
```bash
npm install
createdb gimdb
npm run latest
npm run seed
npm run dev
```

2. Let's run simulator
```bash
npm install
expo start --ios
```
